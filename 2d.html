Begin

This is where I import some dependencies.

Loading Sprites

Loads the spritesheet, which contains the player's animations. It's sliced into 39 columns and 31 rows of sprites. Then define which sprite frames are used for specific animations. For example, the "walk-down" animation uses frames 936 to 939 in a loop at a speed of 8 frames per second.

Setting Background and Loading the Map

Sets the background color of the game window using a hex code. Then load the image.

Scene Setup

Loads a JSON file which contains the map data. Then extracts different layers from the JSON. Finally, adds the map to the game sprite.

Player Setup

The player is defined and uses the sprite sheet then start with the idle animation.

Adding Boundaries and Collisions

Add by using JSON data from map.json. These boundaries are static. Then when the player collides with a boundary, a dialogue pops up.

Spawning Point

The player's starting point is determined by spawnpoints from the JSON data.

Camera

The camera is adjusted to scale with the window size. And the camera also follows the player's world coordinates.

Player Movement

When the left mouse is clicked, the player moves toward the clicked position.

Stop Animation

When the player stops moving , the animation switches back to idle state.

API Setup

We need to have a unique api key to request the OpenWeatherApp API. apiWeb is the base URL for the API request.

DOM Selection

The first one stores input. The button trigger the weather search when a user click.

Asynchronous Weather

The function takes a city name as a parameter, builds a complete API request URL then uses fetch to retrieve the data.

Error Handling

The code checks if the response status from the API is 404(the city not found).

Display Data

The weather information is displayed on the web by updating the inner HTML of specific elements.

Icon Display

Based on the data.weather[0].main (the main weather condition, like "Clouds" or "Clear"), the corresponding weather icon is displayed by updating the weatherIcon.src. .

Show Weather and Hide Error

If the weather data is retrieved, the .weather block is displayed and .error message is hidden.

Handling Button

When search button is clicked, the checkWeather is triggered. And, the value entered in the search input is passed as an argument to the function then fetching and displaying the weather data for that city.
